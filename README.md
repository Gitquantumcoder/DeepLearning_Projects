# 🧠 DeepLearning Projects

A beginner-friendly collection of deep learning notebooks covering image classification, sequence modeling, and neural network experiments.

## ✨ Highlights

- 📚 Multiple hands-on notebooks for CNN, RNN, and deep learning concepts
- 🐱 Cat vs Dog image classifier project
- 🔄 CNN vs RNN comparison experiments
- 📝 Sentiment analysis using recurrent neural networks
- 🧠 Building a brain-inspired neural network exploration

## 🛠️ Tech Stack

- Python
- Jupyter Notebook
- PyTorch
- NumPy
- Pandas
- Matplotlib
- scikit-learn

## 📁 Project Structure

- CNN vs RNN/ - notebook comparing convolutional and recurrent architectures
- CNN_cat_dog_classifier/ - image classification project for cat and dog images
- Nvidia_BuildingABrain/ - neural network exploration notebook
- RNN_sentiment analysis/ - sentiment analysis experiment with RNNs

## 🚀 Getting Started

1. Clone the repository
2. Install the required Python packages
3. Open the notebooks in Jupyter Notebook or JupyterLab
4. Run the cells to explore the projects

## 📦 Notes

This repository intentionally excludes large dataset files and trained model artifacts to keep the project lightweight and Git-friendly.
